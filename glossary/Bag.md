
## Definition

A bag is a file or collection of files used to record and store ROS 2 data, such as messages published on [[Topic]]s.

## Purpose

Bags allow ROS 2 applications to record communication data and replay it later for testing, debugging, analysis, and development.

## Example

- Recording data from the `/number` topic using `ros2 bag record`.
- Replaying the recorded data using `ros2 bag play`.
